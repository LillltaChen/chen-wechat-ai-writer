# 公众号 AI 全自动写作

端到端自动化工作流：选题 → 搜素材 → 写正文 → 生成标题 → 封面提示词 → 排版 → 推送草稿箱。

## 执行流程

按以下 8 步执行。每步可直接调用 AI 生成，或使用 [scripts/wechat_writer.py](scripts/wechat_writer.py) 一键自动化。

### Step 0：选题

用户提供话题、关键词或热点事件。无选题时，生成 5 个候选供选择，每个附"为什么值得写"的理由。

生成 Prompt 示例：
```
你是一名资深公众号编辑，擅长挖掘高潜力选题。
我的账号定位是：[用户定位，如果没给就询问]
目标读者是：[用户受众，如果没给就询问]

请生成 5 个选题，要求：
1. 有痛点或情绪共鸣
2. 有信息差或实用价值
3. 适合目标读者阅读
4. 每个选题给出"为什么值得写"的理由
```

### Step 1：素材搜集

针对选题搜索案例、数据、金句、反常识观点。输出结构化素材包：核心观点 3-5 个、具体案例 2-3 个（带数据）、金句 3-5 条、反常识角度 1-2 个、读者痛点 2-3 个。

### Step 2：爆款结构拆解（可选）

分析同类爆款文章的 3 种常见结构。按「开头→中段→结尾」描述每种结构的适用场景、情绪节奏、字数配比，推荐最适合本文的一种。

### Step 3：生成大纲

基于素材和推荐结构输出详细大纲。标注每部分预计字数、钩子/案例/情绪升华位置。开头 50 字内抓住读者，结尾给出可执行的 Next Step。

### Step 4：写作正文

按大纲写出完整文章。风格要求：口语化、短句、有节奏感、去 AI 味；每段不超过 3 行，多用空行制造呼吸感；案例具体，带数字和细节；关键句加粗（**加粗**）；适当使用「你」拉近距离。Markdown 格式输出。

### Step 5：生成标题

按以下公式各生成 3 个标题：数字法、悬念法、痛点法、反常识法、对话法。从 15 个中选出 Top 3 并附推荐理由，由用户最终选择或自动选 Top 1。

### Step 6：生成封面提示词

基于最终标题和选题，生成英文 AI 封面图提示词。要求：2.35:1 横版比例，现代高级有设计感，画面简洁预留文字区域，适合公众号信息流展示。直接输出可用的 Midjourney / GPT-4o 提示词。

### Step 7：排版转公众号 HTML

将 Markdown 正文转为微信公众号 HTML 格式。要求：h1/h2/h3 用合适字号和加粗；段落行高 1.8，字号 16px；**加粗** 转为 `<strong>`；列表用 `<ul><li>`；整体配色简洁黑灰为主。

### Step 8：推送到草稿箱

调用微信草稿箱 API：
```
POST https://api.weixin.qq.com/cgi-bin/draft/add?access_token=ACCESS_TOKEN
Body:
{
  "articles": [{
    "title": "标题",
    "author": "作者",
    "content": "HTML内容",
    "thumb_media_id": "封面素材ID",
    "show_cover_pic": 1,
    "need_open_comment": 1
  }]
}
```

需先获取 `access_token`（AppID + AppSecret）并上传封面图获取 `thumb_media_id`。

## 自动化脚本

使用 [scripts/wechat_writer.py](scripts/wechat_writer.py) 一键执行全流程：

1. 安装依赖：`pip install requests`
2. 准备封面图：放 `cover.jpg` 在同目录
3. 配置脚本顶部 4 个 Key：OpenAI API Key、Serper API Key（可选）、微信 AppID、微信 AppSecret
4. 运行：`python scripts/wechat_writer.py`

## 微信 API 凭证获取

1. 登录 [mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 设置与开发 → 基本配置 → 获取 **AppID** 和 **AppSecret**
3. 添加 IP 白名单（运行脚本的服务器/电脑公网 IP）
4. 个人订阅号即可使用，无需认证

## 进阶优化方向

| 优化点 | 说明 |
|--------|------|
| 标题 A/B 测试 | 生成 2 个标题，先发朋友圈测点击率 |
| 封面自动生成 | 接 Midjourney / GPT-4o Image API |
| 文中插图自动插入 | 关键词触发时自动搜索/生成配图 |
| 定时发布 | 调用微信 API 的发布接口设定时间 |
| 数据回流 | 发布后 24h 自动读数据，写复盘报告 |

**核心原则**：AI 负责写和推，人只负责选题和点群发。
